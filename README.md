# ouorz
基于 King 主题使用 Vue.js 重构修改的 WordPress 个人博客主题

<br/>

### 修改内容
+ 文章内链
+ 评论功能(已关闭)
+ 特殊文章列表样式
+ 全站Vue.js重构
+ 首页分类/标签
+ 文章列表无限加载
+ 使用 Artalk 作为评论系统(https://github.com/qwqcode/Artalk)
